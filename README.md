# Legend of Zelda 

Legend of zelda game in java and javascript different languages. Each version implementing a new game mechanic. Object-orientated programming.

## java_link

### How to run

Using Windows:

```
git clone https://github.com/MBowsman/legend_of_zelda
cd legend_of_zelda
/build.bat
```
### Keys
* 'Arrow keys' to move link
* 'E' Edit Mode
* 'A' Add Mode
* 'R' Remove Mode
* 'S' Save
* 'L' Load
* 'Q' Quit

### Features
* Build map using edit mode
* Collision detection

## javascript_link

### How to run
1) Run main file
Using Windows:
```
git clone https://github.com/MBowsman/legend_of_zelda
cd legend_of_zelda
main.html
```
2) Drag map.json from repository into box 

### Keys
* 'Arrow keys' to move link
* 'Space' for boomerang

### Features
* Throw boomerangs to collect gems
* Gem counter
* JSON map file
* Camera shift
* Collision detection
