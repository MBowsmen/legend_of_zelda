# Legend of Zelda 

Legend of zelda game in java and javascript different languages. Each version implementing a new game mechanic. Object-orientated programming.

## java_link

### How to run

Using Windows:

```
git clone https://github.com/MBowsman/legend_of_zelda
cd legend_of_zelda
/build.bat
```

## javascript_link

### How to run

Using Windows:

```
git clone https://github.com/MBowsman/legend_of_zelda
cd legend_of_zelda
main.html
```
